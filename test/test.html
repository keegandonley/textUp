Basic tests for verifying the newest version of the textUp library. All tests are run using the most recent version on gitHub. Alternately, the tests can be run using a local version of the library at /test/test.html
This html file includes the jQuery library, which is required for using the styling features (and in the future, animation). For this reason, in a local environment where jQuery has been removed some tests may fail.
